# QuickBites Frontend

This is the Vite + React frontend for the QuickBites food delivery platform.

## Setup

1. Install dependencies: `npm install`
2. Run locally: `npm run dev`

## Notes

- The app entry point is `src/main.jsx`
- The main component is `src/App.jsx`
- Tailwind CSS is configured in `tailwind.config.js`
- Global state is initialized in `src/app/store.js`
- App routes are defined with `react-router-dom`
- Toast notifications are enabled with `react-toastify`
